# Python Learning

## Mutable and Immutable data types 
  
  ## Mutable (jo change ho sakta hai )
  * list 
  * set 
  * dictionary
  * bytearray
  * array

  ## Immutable (jo change nahi hota balki dusri object ban jati hai )

  * integers 
  * floating - point numbers 
  * boolean 
  * strings 
  * frozen set 
  * bytes
